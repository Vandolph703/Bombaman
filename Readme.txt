Instructions for running the code:

showing how to run the code.

Game layout:
- 9x9 grid
- Blue blocks are destructible.
- Green blocks are invincible.

Instructions for playing the game:
IMPORTANT NOTE: In order to see the changes, you must left click the screen.
This is an annoying bug that occured near the end stages of implementation.

W - move up
S - move down
A - move left
D - move right
Space- place bomb (red square)
K - Detonate bomb
